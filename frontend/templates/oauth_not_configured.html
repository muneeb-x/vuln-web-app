Security Vulnerability Lab

Google login isn't configured

"Continue with Google" needs Google OAuth credentials, which this copy of the app doesn't have set yet.

To enable it, an operator must create a Google OAuth client, then copy .env.example to .env and fill in GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET. The full step-by-step walkthrough is in the project docs (docs/continue-with-google-explained.md) and the README's "Continue with Google — Setup" section.

You can still sign in with a username and password in the meantime.
