Our Mission

This platform is designed as an educational tool for learning about common web security vulnerabilities. By exploring and exploiting these intentional flaws, you will gain practical understanding of how real attacks work and how to defend against them. All vulnerabilities are deliberately implemented for educational purposes only.

VULNERABILITIES TO DISCOVER

SQLi

SQL Injection

Raw SQL query construction allows authentication bypass through malicious input in login fields.

XSS

Stored XSS

Malicious scripts stored in the database execute automatically when the dashboard page loads.

XSS

Reflected XSS

User input reflected in search results without escaping allows one-time script execution.

Session

Session Hijacking

Weak hardcoded secret key allows session cookie theft and reuse across browsers.

Crypto

Weak Password Storage

MD5 hashing without salt enables rainbow table attacks for password recovery.

Exposed

Exposed Database

Database file downloadable via unauthenticated endpoint exposes all user data.

Brute

No Rate Limiting

Unlimited login attempts with no throttling enable automated brute force attacks.

CSRF

CSRF

No token validation on form submissions allows cross-site request forgery attacks.
